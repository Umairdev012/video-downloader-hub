# YouTube Video Downloader

A modern and responsive YouTube Video Downloader UI built using HTML, CSS, and JavaScript.

## Features

* YouTube URL Validation
* Video ID Extraction
* Thumbnail Preview
* Responsive Design
* Loader Animation
* Error Handling
* Download Options UI
* Mobile Friendly Layout
* Modern Dark Theme

## Current Version

This version is a frontend preview version.

It can:

* Validate YouTube URLs
* Extract Video IDs
* Display Video Thumbnails
* Render Video Preview Cards

It cannot directly download YouTube videos because browsers do not allow direct video extraction from YouTube.
